An Image

This is an example page showing Lightweight Theme (LT) by Jan-Gerd Tenberge. LT is focussed on readable typography, provided by the Arvo and Noticia Text fonts, and beautifully huge images. The width of each article is exactly 800px and your images, source code, or maps should fill it right to left to stand out. LT supports a number of features including retina displays. Specify an optional high-resolution source for your images with <img data-retinasrc=""> and LT will automatically use it on high-resolution screens.

Syntax Highlighting

The replacement of images is done in just a few lines of JavaScript shown below. Syntax highlighting for all widely used programming languages is provided by highlight.js and uses the Inconsolata font by Raph Levien. 


  var images = document.getElementsByTagName("img");
  if (window.devicePixelRatio > 1.5) {
    for (var i=0; i<images.length; i++) {
      if (images[i].getAttribute('data-retinasrc')) {
        images[i].src = images[i].getAttribute('data-retinasrc');
      }
    }
  }

As you see, there is no need for block elements like images or source code to be at the very top of an article. They may be placed anywhere within and you can easily have multiple source code environments within a single article. Highlight.js will automatically figure out the language and correct highlight mode for each of them, here is an example in C


  #include <stdio.h>

  int main(void) {
    printf("Hello, World\n");
    return 0;
  }

For more esoteric languages you may need to provide your own version of highlight.js, you can choose from a wide range of languages and build your own package right here at Software Maniacs.
